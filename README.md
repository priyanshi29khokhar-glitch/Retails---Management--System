# Retail Management System

## Project Overview
Developed a Retail Management System using SQL to manage and analyze retail store sales data.

## Features
- Store sales records
- Calculate total revenue
- Calculate total quantity sold
- Count total orders
- Category-wise revenue analysis
- Average product price calculation

## SQL Concepts Used
- CREATE TABLE
- INSERT INTO
- SELECT
- SUM()
- COUNT()
- AVG()
- GROUP BY

## Tools Used
- MySQL
- GitHub
